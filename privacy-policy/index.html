SIS CSV Autofill Chrome Extension – Privacy Policy

Effective date: 12/24/2025

1. Introduction

The SIS CSV Autofill Chrome extension is designed to assist teachers in filling grades from a locally uploaded CSV file into their institution’s Student Information System (SIS) pages.

This extension does not collect, store, or transmit any user data. All processing occurs locally in the user’s browser, and the user manually submits grades in the SIS interface.

2. Data Collection

The extension does not collect any of the following information:

  • Personally identifiable information (name, student IDs, email, etc.)
  • Health information
  • Financial or payment information
  • Authentication credentials or passwords
  • Personal communications (emails, chats, messages)
  • Location data
  • Web browsing history
  • User activity such as clicks, scrolls, or keystrokes
  • Website content beyond the active SIS tab (for autofill purposes only)

3. How Data Is Used

  • No data is sent, shared, or stored externally.
  • All operations occur locally in the user’s browser.
  • The extension only accesses the currently active SIS page when the teacher clicks “Apply Grades”.

4. Data Sharing

  • The extension does not sell, transfer, or disclose any information to third parties.
  • No information is used for purposes unrelated to the extension’s single purpose of autofilling grades.
  • The extension does not collect data for creditworthiness, lending, or profiling.

5. Security

  • All processing occurs locally.
  • No remote servers are contacted.
  • No external scripts or dynamic code is executed.
